# Install Notes

## What this pack does

This pack replaces the repo's old MIT/open-source framing with:

- a public-source README
- a free public license for smaller/private users
- paid commercial requirements for governments and large enterprises
- a $999 Developer Welcome Kit page and terms
- cross-links between README, licensing, buy page, and commercial terms

## Files included

- `README.md`
- `LICENSE`
- `NOTICE`
- `COMMERCIAL-LICENSE.md`
- `WELCOME-KIT.md`
- `licensing.md`
- `buy-welcome-kit.md`

## Before publishing

Replace these placeholders:

- `REPLACE_WITH_CHECKOUT_LINK`
- `REPLACE_WITH_PHONE_NUMBER`
- `licensing@hashsite.org` if you plan to use a different address

## Suggested first steps

1. Copy these files into the repo root.
2. Commit and push.
3. Create a hosted checkout page for the $999 Welcome Kit.
4. Replace the checkout and phone placeholders.
5. Test:
   - the repo root README
   - `/licensing/`
   - `/buy/`

## Suggested commit message

`Replace MIT/open-source framing with public-source license stack and Welcome Kit pages`
